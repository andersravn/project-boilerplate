## Quick Start

### 1. Clone the Repository

<CodeBlock language="bash">
{`git clone <repository-url> my-project
cd my-project`}
</CodeBlock>

### 2. Install Dependencies

<CodeBlock language="bash">
{`npm install
# or
yarn install
# or
pnpm install`}
</CodeBlock>

### 3. Start Development Server

<CodeBlock language="bash">
{`npm run dev`}
</CodeBlock>

Your application will be available at `http://localhost:3000`.

### 4. Complete Core Installation

Before using components, set up the required dependencies and utilities:

Visit the [Setup Guide](/docs/components/setup) to set up the core utilities (cva, clsx, tailwind-merge) and CSS variables needed by all components.

### 5. View Documentation

Visit `http://localhost:3000/docs` to see this documentation and component library.

<Alert variant="warning" title="Documentation Access">
  The `/docs` route is only available in development mode for security reasons.
</Alert>

## Configuration

### Tailwind CSS

The project uses Tailwind CSS for styling. Customize your design system in `tailwind.config.js`:

<CodeBlock filename="tailwind.config.js">
{`module.exports = {
  content: ['./src/**/*.{js,ts,jsx,tsx,mdx}'],
  theme: {
    extend: {
      colors: {
        // Add your brand colors
        brand: {
          50: '#f0f9ff',
          500: '#3b82f6',
          900: '#1e3a8a',
        },
      },
      fontFamily: {
        // Add your custom fonts
        sans: ['Inter', 'system-ui', 'sans-serif'],
      },
    },
  },
  plugins: [],
};`}
</CodeBlock>

### TypeScript

The project is fully typed with TypeScript. Key configuration files:

- `tsconfig.json` - TypeScript compiler options
- `next-env.d.ts` - Next.js type definitions
- `src/types/` - Custom type definitions

### shadcn/ui

Component configuration is managed in `components.json`:

<CodeBlock filename="components.json">
{`{
  "style": "new-york",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "config": "",
    "css": "src/app/globals.css",
    "baseColor": "neutral",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils"
  }
}`}
</CodeBlock>

## Adding Components

### From shadcn/ui

Add pre-built components using the CLI:

<CodeBlock language="bash">
{`# Add a single component
npx shadcn@latest add card

# Add multiple components
npx shadcn@latest add card dialog sheet

# List available components
npx shadcn@latest add`}
</CodeBlock>

### Custom Components

Create your own components in `src/components/ui/`:

<CodeBlock filename="src/components/ui/custom-component.tsx">
{`import React from 'react';
import { cn } from '@/lib/utils';

interface CustomComponentProps {
  children: React.ReactNode;
  variant?: 'default' | 'secondary';
  className?: string;
}

export default function CustomComponent({ 
  children, 
  variant = 'default', 
  className 
}: CustomComponentProps) {
  return (
    <div 
      className={cn(
        'base-styles',
        variant === 'secondary' && 'secondary-styles',
        className
      )}
    >
      {children}
    </div>
  );
}`}
</CodeBlock>

## Customizing for Your Brand

### 1. Colors

Update CSS variables in `src/app/globals.css`:

<CodeBlock filename="src/app/globals.css">
{`:root {
  --background: 0 0% 100%;
  --foreground: 222.2 84% 4.9%;
  --primary: 221.2 83.2% 53.3%;        /* Your brand primary */
  --primary-foreground: 210 40% 98%;
  --secondary: 210 40% 96%;
  --secondary-foreground: 222.2 84% 4.9%;
  /* Add more custom colors */
}`}
</CodeBlock>

### 2. Typography

Add custom fonts and typography scales:

<CodeBlock filename="tailwind.config.js">
{`theme: {
  extend: {
    fontFamily: {
      sans: ['YourFont', 'system-ui', 'sans-serif'],
      heading: ['YourHeadingFont', 'serif'],
    },
    fontSize: {
      'brand-xl': ['3.5rem', { lineHeight: '1.1' }],
      'brand-lg': ['2.5rem', { lineHeight: '1.2' }],
    },
  },
}`}
</CodeBlock>

### 3. Component Variants

Extend component variants to match your design system:

<CodeBlock filename="src/components/ui/button.tsx">
{`const buttonVariants = cva(
  "base-button-styles",
  {
    variants: {
      variant: {
        default: "default-styles",
        // Add your custom variants
        brand: "bg-brand-500 text-white hover:bg-brand-600",
        gradient: "bg-gradient-to-r from-brand-500 to-brand-600",
      },
      // Add custom size variants
      size: {
        default: "h-10 px-4 py-2",
        xs: "h-6 px-2 text-xs",
        xl: "h-14 px-8 text-lg",
      },
    },
  }
);`}
</CodeBlock>

## Deployment

### Vercel (Recommended)

<CodeBlock language="bash">
{`# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Or connect your Git repository at vercel.com`}
</CodeBlock>

### Other Platforms

The project works with any platform that supports Next.js:

- **Netlify**: Configure build command as `npm run build`
- **AWS Amplify**: Use the Next.js preset
- **Railway**: Connect your repository and deploy
- **Digital Ocean**: Use the App Platform with Node.js

<Alert title="Environment Variables">
  Remember to set your environment variables in your deployment platform, especially for CMS integrations.
</Alert>
