# Setup

Set up the required dependencies and utilities for the Vertica UI component library.

## Core Dependencies

These dependencies are required for most components in the library:

<CodeBlock language="bash">
{`npm install class-variance-authority clsx tailwind-merge`}
</CodeBlock>

### What these do:

- **`class-variance-authority`** - Type-safe component variants with cva()
- **`clsx`** - Utility for constructing className strings conditionally  
- **`tailwind-merge`** - Merge Tailwind CSS classes without style conflicts

## Utility Function

Create the `cn` utility function that combines clsx and tailwind-merge:

<CodeBlock filename="src/lib/utils.ts">
{`import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}`}
</CodeBlock>

## TypeScript Configuration

Ensure your `tsconfig.json` includes path mapping for imports:

<CodeBlock filename="tsconfig.json">
{`{
  "compilerOptions": {
    // ...other options
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}`}
</CodeBlock>
