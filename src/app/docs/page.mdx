## Framework Compatibility

### React (Standalone)

Our components work perfectly in any React application:

<CodeBlock filename="App.js">
{`import Button from './components/ui/button';

function App() {
  return (
    <div>
      <h1>My React App</h1>
      <Button onClick={() => alert('Hello!')}>
        Click me
      </Button>
    </div>
  );
}

export default App;`}
</CodeBlock>

### Next.js

Components also work seamlessly with Next.js:

<CodeBlock filename="page.tsx">
{`import Button from '@/components/ui/button';

export default function HomePage() {
  return (
    <main>
      <h1>My Next.js App</h1>
      <Button>Get Started</Button>
    </main>
  );
}`}
</CodeBlock>

## Customization

All components use Tailwind CSS classes and CSS variables for theming. You can customize:

- **Colors**: Modify CSS variables in your `globals.css`
- **Sizes**: Adjust the variants in component files
- **Animations**: Customize transition classes

<CodeBlock filename="globals.css">
{`:root {
  --primary: 222.2 84% 4.9%;
  --primary-foreground: 210 40% 98%;
  --secondary: 210 40% 96%;
  --secondary-foreground: 222.2 84% 4.9%;
  /* Add your custom colors */
}`}
</CodeBlock>
